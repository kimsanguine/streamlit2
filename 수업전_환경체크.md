## 0. 터미널 여는 법 (터미널이 처음이라면)

1. **터미널 열기** — Windows: 시작 메뉴에 `PowerShell` 입력(또는 `cmd`) / Mac: Spotlight(⌘+Space)에 `Terminal` 입력

> 🪟 **Windows 사용자 읽기 규칙(중요)**: 이 교안의 모든 `python3.11 ...` 명령은 Windows에서는 **`py -3.11 ...`**(또는 파이썬이 하나뿐이면 `python ...`)으로 바꿔 실행하세요. 예) `python3.11 -m streamlit run apps/m1_hello.py` → `py -3.11 -m streamlit run apps/m1_hello.py`. 확인: `py -3.11 --version`이 3.11.x를 출력하면 준비 완료.
2. **이 폴더로 이동** — `cd ` 까지 친 뒤, Finder(또는 탐색기)에서 `Streamlit` 폴더를
   터미널 창으로 **드래그**하면 경로가 따옴표째 자동 입력됩니다. 그다음 엔터. (폴더 경로에 공백·한글·
   대괄호가 있어 따옴표가 꼭 필요합니다 — 직접 타이핑하면 틀리기 쉬우니 드래그를 권합니다.)
3. **내 파이썬 확인** — `python3.11 --version` 을 쳐서 `Python 3.11.x`가 나오는지 확인하세요.
   안 나오면 `python --version` / `python3 --version` 으로 3.11대가 어디에 있는지 찾습니다.

## 1. Python 패키지

```bash
python3.11 -m pip install -r requirements.txt
```

- ⏳ **첫 설치는 torch 등 무거운 패키지로 5~15분 걸릴 수 있습니다.** 커서만 깜빡여도 멈춘 게
  아니니 **끄지 말고 기다리세요**(중간에 Ctrl+C로 끊으면 반쯤 설치돼 오히려 더 복잡해집니다).
- 검증 환경: `python3.11` (실측 버전은 `requirements.txt` 주석 참고)
- `pip`가 아니라 `python3.11 -m pip`로 실행하는 이유: 파이썬이 여러 개인 PC(특히 Mac)에서
  `pip`만 쓰면 엉뚱한 파이썬에 깔려, 나중에 `streamlit`이 `ModuleNotFoundError`로 안 뜰 수 있습니다.

## 2. 모델 사전 다운로드 (ST4 필수)

```bash
python3.11 prerequisite.py
```

- MobileNetV2(약 13.6MB)와 KoELECTRA-small NSMC(약 56.5MB)를 미리 캐시합니다.
- 처음 실행 시 약 1~2분, 이미 캐시돼 있으면 수십 초 안에 끝납니다(재실행 안전 — 몇 번 돌려도 됩니다).
- **왜 미리 받아야 하는가**: 50명이 수업 중 동시에 모델을 내려받으면 강의실 네트워크가
  버티지 못합니다. ViT-base(346MB)급 모델을 실수로 동시에 받으면 수업이 통째로 멈춥니다 —
  ST4는 그래서 일부러 작은 모델(MobileNetV2·KoELECTRA-small)만 씁니다.
- 스크립트가 마지막에 KoELECTRA 테스트 추론 결과를 출력합니다. `label`이 `'positive'`가 아니라
  `'1'`/`'0'`으로 나오는 것이 정상입니다 — ST4에서 이 부분을 직접 다룹니다.

## 3. (선택) 로컬 임베딩 — Ollama nomic-embed-text (ST4)

ST4의 PDF RAG 챗봇은 임베딩(embedding) 검색을 **local(Ollama)·openrouter·openai** 3곳 중 골라 쓸 수 있습니다.
로컬 무료 임베딩을 써보고 싶다면 [Ollama](https://ollama.com)를 설치한 뒤 아래 한 줄을 미리 받아두세요.

```bash
ollama pull nomic-embed-text
```

- 약 274MB, 가역적(안 쓰면 그냥 디스크에 남아만 있고 특강에 영향 없음)입니다.
- **안 받아도 됩니다** — openrouter·openai 임베딩(키가 있다면)이나 TF-IDF 폴백으로 수업을 그대로 따라갈 수 있습니다.
- 이미 `ollama pull hermes3:8b`(LLM용)를 받아뒀다면 Ollama 설치는 끝난 상태이니 이 한 줄만 추가하면 됩니다.

## 4. 팔머펭귄 데이터셋 (ST3)

`seaborn.load_dataset("penguins")`는 첫 호출 시 인터넷에서 CSV를 내려받습니다(수백 KB, 매우 작음).
prerequisite.py에는 포함하지 않았으니, 아래 한 줄로 미리 캐시해두면 ST3 시작이 더 매끄럽습니다.


## 5. 실행 기준

- 노트북 셀 안에서 `streamlit run ...`을 직접 실행하지 않습니다. 서버가 멈추지 않아 셀이
  끝나지 않습니다 — 반드시 **터미널**에서 실행합니다.
- 기본 실행: `streamlit run apps/m3_penguins.py`
- `ModuleNotFoundError` 등으로 실행이 안 되면 파이썬이 여러 개인 PC입니다. 노트북과 같은
  파이썬으로 고정해 `python3.11 -m streamlit run apps/m3_penguins.py`로 실행하세요(**폴백 필수**).
- 배포 전(ST6)에는 모든 노트북을 `Restart & Clear Output` 상태로 정리합니다.
